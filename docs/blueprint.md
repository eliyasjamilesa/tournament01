# **App Name**: IgniteArena

## Core Features:

- Mode Discovery Dashboard: A visually immersive home screen allowing players to browse categories like BR E-sports, Clash Squad, and Lone Wolf matches with real-time match counts.
- AI Tactical Scout: An AI-powered tool that analyzes selected match types and provides specific drop-zone and weapon strategies based on current game metadata.
- One-Tap Tournament Entry: Simplified registration flow where users can submit their player ID and squad details to join upcoming matches.
- Live Match Tracking: Display currently active tournaments and live lobby statuses in the 'Play' tab as seen in the reference material.
- Competitive Result Feed: A dedicated view for tracking rankings, winners, and recent performance of completed tournament sessions.
- Integrated Player Profile: Personalized user section to manage registered tournaments, 'My Matches' history, and basic stats.

## Style Guidelines:

- Primary Color: Volcanic Red (#FF4119) used for key interactions and highlighted mode headers, evoking high-stakes energy.
- Background Color: Obsidian Coal (#141110), a deeply desaturated dark hue with a slight warm undertone to complement the fire theme.
- Accent Color: Magenta Flash (#E61962), placed 30 degrees to the left of the primary red on the color wheel to provide sharp visual contrast for status icons.
- Headline Font: 'Space Grotesk' for a computerized, technical gaming aesthetic. Body Font: 'Inter' for clean readability of tournament stats and lists.
- Solid, chunky icons for bottom navigation and game modes, utilizing red glowing effects similar to the 'fire' branding in the prompt.
- High-contrast card-based grid layout with textured 'magma' background overlays on tournament categories.
- Pulsing red glow on active matches to draw immediate user attention to live gaming opportunities.
